Why Serratus?

The world needs us. We are currently in a pandemic and we are ill-equiped. The quality of the sequencing data available for SARS-Cov-2 (Covid-19) is not in good condition, with only 12% of coronaviruses having complete genomes. It is also likely that many more viruses in the same family have yet to be discovered. Serratus is here to help, but how?

NCBI SRA Database

The NCBI SRA database contains over 1 million RNA-seq libraries amounting to a whopping 5.72 petabytes (yes, that prefix is right, peta-bytes) of data. Within that data lies possible evidence of the origin of Covid-19. It is our goal to reanalyze all of these sequences utilizing our ultra-high throughput analysis pipeline hosted on Aamazon Web Service's (AWS) Cloud Computing to find this evidence.

Cloud Architecture

Since there is a lot of data, we need a lot of computers. AWS allows us to spin up hundreds of them at a single time to process data in parallel, utilizing the same bioinformatics pipeline on each of them. Serratus is currently operational, scaling up to 250 nodes with real time tracking and cluster performance analysis, but we are not stopping there...

CoV Sequence Database

Once serratus is done, data is processed, cleaned and analyzed. Then, all resulting alignment files will be stored in a free, immediate access public database. It is our goal to provide honest and accurate data in order to continue the collaborative effort in fighting Covid-19.
